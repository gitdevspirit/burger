# Burger v1.0.0 - Initial Release 🍔

## What's New

This is the first release of Burger - a clean, minimal Minecraft 1.21 Fabric client for Hypixel Skyblock!

### Features

- ✅ Clean mod framework ready for development
- ✅ Configuration system with JSON support
- ✅ Example welcome message feature
- ✅ Example mixin (Title screen)
- ✅ Utility classes for common operations
- ✅ Hypixel/Skyblock detection
- ✅ Comprehensive documentation

### What's Included

- Main mod class with tick handling
- Config system (BurgerConfig)
- Utils class with helper methods
- Example feature showing how to add features
- Example mixin showing how to inject code
- 5 documentation files

### Installation

1. Download `burger-1.0.0+1.21.jar` from this release
2. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21
3. Download [Fabric API](https://modrinth.com/mod/fabric-api/versions?g=1.21) 0.102.0+1.21 or higher
4. Place both JARs in your `.minecraft/mods` folder
5. Launch Minecraft 1.21 with Fabric profile

### Requirements

- Minecraft 1.21
- Fabric Loader 0.16.5+
- Fabric API 0.102.0+1.21
- Java 21+
