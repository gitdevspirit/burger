# Changelog

All notable changes to Burger will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-20

### Added
- Initial release of Burger mod
- Basic mod structure and framework
- Configuration system with JSON support
- Welcome message feature (example)
- Title screen mixin (example)
- Utility classes for common operations
- Hypixel and Skyblock detection
- Complete documentation (5 files)
- Custom burger icon

### Technical Details
- Minecraft 1.21 support
- Fabric Loader 0.16.5
- Fabric API 0.102.0+1.21
- Java 21 compatibility
- MIT License

### Developer Features
- Clean, well-documented codebase
- Example feature implementation
- Example mixin implementation
- Config system ready to extend
- GitHub Actions for automatic builds
